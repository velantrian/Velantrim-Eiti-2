## 🌟 Что такое VELANTRIM EITI?

VELANTRIM EITI — это монолитный AI-ассистент нового поколения, упакованный в **единственный HTML-файл**. Никакого Node.js, никакого Python, никакого бэкенда. Просто открой файл — и работай.

Внутри — полноценная архитектура с многоуровневой памятью, reasoning-движком, базой знаний, музыкальным плеером, заметками, файловым менеджером и календарём. Всё в одном файле.

## ⚡ Ключевые особенности

| | Особенность | Описание |
|---|---|---|
| 🧠 | **DAAD FractalMemory** | Трёхуровневая архитектура памяти (L0 / L1 / L2) |
| 🔌 | **Zero Dependencies** | Один `.html` файл, никаких установок |
| 📡 | **PWA + Offline** | Работает без интернета после первой загрузки |
| 🤖 | **Мульти-провайдер AI** | DeepSeek · Gemini · Grok · OpenRouter · DDG |
| 🔍 | **FTS5 поиск** | Полнотекстовый поиск с BM25-ранжированием на SQLite WASM |
| 🌐 | **Telegram Web** | Встроенный таб с web.telegram.org |
| 🎨 | **Темы и дизайны** | 10+ тем, скевоморф, стекло, минимализм |
